# Быстрый старт - Форма 052/у

## Как открыть форму

1. Запустите проект (если еще не запущен):
```bash
npm run dev
```

2. Откройте в браузере:
```
http://localhost:5173/form052
```

## Основные возможности

### ✅ Реализовано

- **Общая часть** - полностью функциональный раздел с:
  - Паспортными данными (14 полей)
  - Минимальными медицинскими данными (14 полей)
  - Динамическим добавлением аллергических реакций
  - Всеми необходимыми полями согласно форме 052/у

### ⏳ В разработке

Остальные разделы имеют заглушки и будут реализованы по мере необходимости.

## Использование

1. **Заполнение формы**:
   - Переключайтесь между разделами через табы
   - Заполняйте поля в режиме редактирования
   - Нажимайте "Сохранить" для сохранения данных

2. **Просмотр данных**:
   - Нажмите "Режим просмотра" для просмотра заполненных данных
   - Все поля будут доступны только для чтения

3. **Печать**:
   - Нажмите кнопку "Печать" для открытия диалога печати
   - Форма оптимизирована для печати на A4

## Сохранение данных

Данные сохраняются в `localStorage` браузера с ключом `form052_data`.

При следующем открытии формы данные автоматически загрузятся.

## Интеграция

Когда форма будет готова, она может быть интегрирована в основной проект через:

```typescript
import Form052Editor from './components/Form052Editor';

// Использование
<Form052Editor
  initialData={data}
  mode="edit"
  onSave={handleSave}
  onPrint={handlePrint}
/>
```

## Структура

- `components/Form052Editor.tsx` - главный компонент
- `components/form052/` - компоненты разделов
- `types/form052.ts` - типы данных
