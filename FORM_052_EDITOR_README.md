# Форма № 052/у - Редактор медицинской карты амбулаторного пациента

## Описание

Полнофункциональный редактор формы № 052/у "Медицинская карта амбулаторного пациента" с удобным интерфейсом для заполнения и просмотра всех разделов формы.

## Доступ

Форма доступна по отдельному URL: **`/form052`**

Например:
- `http://localhost:5173/form052` (локальная разработка)
- `https://yourdomain.com/form052` (продакшн)

## Структура формы

Форма состоит из следующих разделов (табов):

### 1. Общая часть ✅
- **Паспортные данные**: ИИН, ФИО, дата рождения, пол, возраст, национальность, гражданство, адрес, место работы, должность, образование, страховка, социальный статус, повод обращения
- **Минимальные медицинские данные**: группа крови, резус-фактор, аллергические реакции, физиологическое состояние, скрининг новорожденных, вредные привычки, профилактические мероприятия, история болезней, текущие проблемы, динамическое наблюдение, группа инвалидности, принимаемые лекарства, антропометрические данные, оценка риска падения, оценка боли

### 2. План работы с семьей (в разработке)
- Индивидуальный план работы с семьей в организациях ПМСП
- Данные детей и членов семьи
- Представители внешних организаций

### 3. Патронаж новорожденного (в разработке)
- Вкладной лист 1: Патронаж новорожденного (младенца) на дому
- Физический осмотр
- Оценка кормления
- Оценка развития

### 4. Оценка развития ребенка (в разработке)
- Вкладной лист 2: Оценка развития ребенка
- Физическое развитие
- Психомоторное развитие
- Оценка ухода

### 5. Рекомендации (в разработке)
- Вкладной лист 3: Заключение и рекомендации
- Консультации специалистов

### 6. Динамическое наблюдение (в разработке)
- Вкладной лист 4-5: Динамическое (диспансерное) наблюдение
- Запись о леченном случае
- Протокол операции/процедуры/афереза

### 7. Профилактические мероприятия (в разработке)
- Вкладной лист 6-7: Профилактические мероприятия
- Вакцинация
- Диагностические исследования

### 8. Осмотр при жестоком обращении (в разработке)
- Вкладной лист 8-9: Карта осмотра при обращении по поводу телесных повреждений
- Описание травм
- Психологическое состояние

### 9. Сурдологический осмотр (в разработке)
- Вкладной лист 9: Карта осмотра сурдологического пациента
- Диагностика нарушений слуха
- Слуховые аппараты и импланты

## Функциональность

### Режимы работы

1. **Режим редактирования** (по умолчанию)
   - Все поля доступны для редактирования
   - Кнопка "Сохранить" активна
   - Данные сохраняются в localStorage

2. **Режим просмотра**
   - Все поля только для чтения
   - Удобный просмотр заполненных данных
   - Кнопка переключения в режим редактирования

### Сохранение данных

- Данные автоматически сохраняются в `localStorage` браузера при нажатии кнопки "Сохранить"
- Ключ сохранения: `form052_data`
- Формат: JSON

### Печать

- Кнопка "Печать" открывает стандартное диалоговое окно печати браузера
- Форма оптимизирована для печати на формате A4

## Использование

### Заполнение формы

1. Откройте форму по адресу `/form052`
2. Переключайтесь между разделами через табы вверху
3. Заполните необходимые поля
4. Нажмите "Сохранить" для сохранения данных

### Просмотр данных

1. Откройте форму по адресу `/form052`
2. Нажмите "Режим просмотра" для переключения
3. Просматривайте заполненные данные

### Загрузка сохраненных данных

Данные автоматически загружаются из `localStorage` при открытии формы.

## Разработка

### Добавление нового раздела

1. Создайте компонент в `components/form052/`
2. Добавьте таб в `Form052Editor.tsx`
3. Добавьте обработку в `renderTabContent()`
4. Обновите типы в `types/form052.ts` при необходимости

### Стилизация

Форма использует Tailwind CSS с кастомными цветами:
- Основной цвет: `slate-900`
- Акцент: `blue-600`
- Фон: `slate-50`
- Границы: `slate-200`
