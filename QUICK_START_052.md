# Быстрый старт - Форма 052/у

## Для врачей

### 1. Вход в систему
- Войдите в систему как врач
- Вы попадете в личный кабинет врача

### 2. Проведение осмотра
1. В маршрутном листе выберите сотрудника
2. Заполните форму осмотра:
   - **Жалобы** - что беспокоит пациента
   - **Объективный осмотр** - данные физикального обследования
   - **Диагноз** - поставленный диагноз
   - **Заключение** - ваше заключение по осмотру
   - **Рекомендации** - рекомендации пациенту
   - **Годен к работе** - отметьте, если годен
3. Нажмите "Сохранить осмотр"

### 3. Для профпатолога (председателя комиссии)
После завершения всех осмотров:
1. Нажмите кнопку "Выставить заключение" у сотрудника
2. Заполните финальное заключение:
   - **Статус** - годен/не годен/наблюдение
   - **Диагноз** - итоговый диагноз
   - **Рекомендации** - рекомендации комиссии
   - **Ограничения** - если есть ограничения к работе
   - **Дата следующего осмотра** - когда проходить следующий осмотр
3. Нажмите "Сохранить заключение"

## Для сотрудников

### 1. Вход в систему
- Войдите в систему как сотрудник
- Вы увидите свою амбулаторную карту

### 2. Просмотр карты
- **Паспортная часть** - ваши личные данные
- **Маршрутный лист** - список врачей для осмотра
- **Лабораторные исследования** - необходимые анализы
- **Осмотры врачей** - результаты осмотров
- **Финальное заключение** - итоговое решение комиссии

### 3. Печать формы 052/у
1. Нажмите кнопку "Печать формы 052/у"
2. Откроется окно с документом
3. Используйте Ctrl+P или кнопку печати браузера
4. Выберите принтер или "Сохранить как PDF"

## Для администраторов клиники

### 1. Добавление врачей
1. Перейдите в раздел "Врачи"
2. Добавьте врачей с указанием специальности
3. Обязательно добавьте профпатолога (председателя комиссии)

### 2. Создание договора
1. Создайте договор с организацией
2. Загрузите контингент сотрудников
3. Утвердите календарный план
4. Система автоматически создаст маршрутные листы для врачей

### 3. Мониторинг
- Отслеживайте прогресс осмотров
- Проверяйте заполнение амбулаторных карт
- Контролируйте выставление финальных заключений

## Структура формы 052/у

Итоговый документ содержит:

1. **Заголовок**
   - Название формы
   - Номер карты
   - Данные клиники

2. **Паспортная часть**
   - ФИО, дата рождения, пол
   - Адрес, телефон
   - Место работы, должность
   - Вредные факторы

3. **Анамнез** (если заполнен)
   - Хронические заболевания
   - Перенесенные заболевания
   - Аллергии
   - Вредные привычки

4. **Антропометрия** (если заполнена)
   - Рост, вес, ИМТ
   - Артериальное давление
   - Пульс

5. **Лабораторные исследования** (если заполнены)
   - Анализы крови
   - Анализы мочи
   - ЭКГ, флюорография
   - Другие исследования

6. **Осмотры врачей-специалистов**
   - Для каждого врача:
     - Жалобы
     - Объективный осмотр
     - Диагноз
     - Заключение
     - Рекомендации
     - Годность к работе

7. **Заключение врачебной комиссии**
   - Итоговый статус (годен/не годен/наблюдение)
   - Диагноз
   - Рекомендации
   - Ограничения
   - Дата следующего осмотра
   - Подпись председателя комиссии

## Частые вопросы

### Когда создается амбулаторная карта?
Карта создается автоматически при первом осмотре любого врача.

### Можно ли редактировать осмотр?
Да, врач может открыть сотрудника снова и изменить данные осмотра.

### Когда доступно финальное заключение?
Финальное заключение можно выставить только после завершения всех осмотров всеми врачами.

### Как сохранить форму 052/у в PDF?
При печати выберите "Сохранить как PDF" вместо принтера.

### Можно ли добавить дополнительные поля?
Да, обратитесь к разработчику для расширения структуры данных.

## Техническая поддержка

При возникновении проблем:
1. Проверьте подключение к интернету
2. Обновите страницу (F5)
3. Очистите кэш браузера
4. Проверьте консоль браузера (F12)
5. Обратитесь к администратору системы

## Безопасность

- Все данные защищены и хранятся в Firebase
- Доступ только для авторизованных пользователей
- Врачи видят только своих пациентов
- Сотрудники видят только свою карту
- Данные передаются по защищенному соединению
